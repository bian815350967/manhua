```

### Compiles and hot-reloads for development

npm run serve
```

```
npm run build
```

### 目录结构
```
  .
  ├── dist                             // 项目build打包目录
  ├── public                           // 项目入口文件
  ├── src                              // 生产目录
  │   ├── assets                       // css iconfont字体图标 和图片资源
  │   ├── components                   // 各种组件
  │   ├── pages                        // 各种页面
  │   │    ├── details.vue             // 帖子详情    
  │   │    ├── home.vue                // 首页    
  │   │    ├── login.vue               // 登录    
  │   │    ├── message.vue             // 消息    
  │   │    ├── personalCenter.vue      // 个人中心    
  │   │    └── publish.vue             // 发布    
  │   ├── router router.js             // 路由配置
  │   ├── app.vue                      // 根组件
  │   └── main.js                      // Webpack 预编译入口   
  ├── .gitignore                       //git忽略文件  
  ├── package.json                     // 项目配置文件    
  ├── vue.config.js                    // vue-cli 3.x 配置
  ├── README.md                        // 项目说明文档
  
  ```
